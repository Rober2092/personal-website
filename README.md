## Description
This is my portfolio created with React.js. It contains an About page, Portfolio page, Contact page, and Resume page. The Portfolio page contains a list of six of my favorite applications. The Resume page contains a link to my resume and a list of my proficiencies. In the footer are links to my Github, LinkedIn.
